# jjb-cmd 命令行工具使用指南

## 常见问题

### Node 版本管理工具冲突

**问题**：使用 nvm 管理工具可能会导致 jjb-cmd 版本混乱，命令执行冲突，等无法正确运行。

**建议**：避免使用 nvm 管理 Node 版本，或确保全局安装的 jjb-cmd 版本正确。

### Node 版本兼容性

**问题**：Node 版本大于 v21.0.0 可能存在执行命令报错问题。

**建议**：使用 Node v18.x 或 v20.x 版本，避免使用 v21.0.0 及以上版本。
